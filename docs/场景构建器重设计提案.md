# 场景构建器重设计提案

## 1. 整体架构调整

### 1.1 统一入口设计
```
┌─────────────────────────────────────────────────────────┐
│  场景中心 (SceneHub)                                      │
├─────────────────────────────────────────────────────────┤
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────┐ │
│  │  模板市场    │  │  我的场景    │  │  场景对比分析    │ │
│  │  (12个模板)  │  │  (已创建)    │  │  (多场景对比)    │ │
│  └─────────────┘  └─────────────┘  └─────────────────┘ │
└─────────────────────────────────────────────────────────┘
```

### 1.2 新模式：向导式构建 + 可视化编辑

**向导模式** (适合新手)
- 5步引导：行业选择 → 业务范围 → 合规目标 → 风险等级 → 生成确认
- 智能推荐：根据选择自动推荐模板和COU

**专家模式** (适合高级用户)
- 基于ControlObjective直接选择
- 五维标签可视化配置
- 权重公式实时预览

## 2. 核心功能模块

### 2.1 场景配置中心 (SceneConfigHub)

```typescript
interface SceneConfiguration {
  // 基础信息
  id: string;
  name: string;
  description: string;
  icon: string;

  // 业务画像 (新增)
  businessProfile: {
    industry: Industry[];           // 所属行业
    scale: 'small' | 'medium' | 'large' | 'enterprise';
    regions: Region[];              // 业务地区
    dataTypes: DataType[];          // 处理的数据类型
    userTypes: UserType[];          // 用户群体
  };

  // 合规目标 (新增)
  complianceGoals: {
    primaryGoal: 'baseline' | 'certification' | 'audit' | 'risk-control';
    targetLevel: number;            // 目标合规等级 1-5
    deadline?: Date;                // 合规期限
    budget?: 'low' | 'medium' | 'high';
  };

  // 风险配置 (新增)
  riskProfile: {
    sensitivityLevel: 'low' | 'medium' | 'high' | 'critical';
    riskTolerance: 'strict' | 'balanced' | 'relaxed';
    priorityAreas: string[];        // 重点关注领域
  };

  // 标签配置
  tags: FiveDimensionalTags;

  // 匹配设置
  matchingConfig: {
    minMatchScore: number;
    autoIncludeRecommended: boolean;
    weightAdjustmentStrategy: 'conservative' | 'balanced' | 'aggressive';
  };

  // 生成结果
  generatedCOUs: GeneratedCOU[];
  createdAt: string;
  updatedAt: string;
}
```

### 2.2 智能推荐引擎

```typescript
interface RecommendationEngine {
```

## 5. 技术实现建议

### 5.1 状态管理优化
```typescript
// 使用状态机管理构建流程
type BuilderState =
  | { step: 'idle' }
  | { step: 'basic-info'; data: Partial<SceneConfiguration> }
  | { step: 'business-profile'; data: Partial<SceneConfiguration> }
  | { step: 'tag-config'; data: Partial<SceneConfiguration> }
  | { step: 'matching'; data: Partial<SceneConfiguration>; matches: MatchedCOU[] }
  | { step: 'preview'; data: SceneConfiguration; generated: GeneratedCOU[] }
```
